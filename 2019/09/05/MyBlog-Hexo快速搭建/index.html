博客搭建

1.安装环境 NodeJS + hexo + git

2.gitbash -> hexo init ->hexo s -> hexo n “MyBlog”

简介：

  • hexo init：初始化
  • hexo s ： 启动博客，用于测试
  • hexo n “MyBlog”：新建博客，名称为“MyBlog”
  • hexo clean：清理数据库
  • hexo g：生成博客

3.部署到github

  • 新建github仓库

  • 修改配置文件 _config.yml：

    deploy:
    type: git
    repo: https://github.com/fryddup/fryddup.github.io.git
    branch: master

  • hexo -d 推到远程服务器

此时可以通过 https://fryddup.github.io/ 来访问我的博客

4.修改博客主题

  • git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia
  • 修改配置文件： tmemes：yilia
